\chapter{Analysis of differences} \label{Chp:Verschil}

Here we compare the results of \dfastmi using the new \emph{stepped-hydrograph} approach with the results of the classic 3-discharge approach of \dfmi 2 / WAQMORF.
The purpose of this comparison is to verify that the results are similar when both methods are used as intended.
For this purpose the following cases are currently used:

\item \nameref{Sec:DeLymen}
\end{itemize}

The additional simulations required for the new \emph{stepped-hydrograph} method have been carried out using the same schematisation and software versions as used in the original analysis.

\section{Gendtse Waard}\label{Sec:GendtseWaard}

\emph{The input files of this case are included in the distribution under \file{examples/03 - Gendtse Waard}.}

This concerns a secondary channel located at the "Gendtse Waard" along the Bovenwaal, a stretch of the river Waal.
This case is also used for the backward compatibility tests, see \autoref{Sec:GendtseWaard_backward}.
In the backward compatibility mode \dfmi uses the flow fields at 3000, 4000, and \SI{6000}{\metre\cubed\per\second} at Lobith.
For the new analysis, this list is extended with 1300, 2000, and \SI{8000}{\metre\cubed\per\second}.
Furthermore, the bed celerity has changed.
In the backward compatibility mode the algorithms use for the Boven-Waal \SI{0.81}{\kilo\metre\per\year} at a discharge of \SI{3000}{\metre\cubed\per\second}, and \SI{3.65}{\kilo\metre\per\year} at 4000 and \SI{6000}{\metre\cubed\per\second}.
The celerity at the Boven-Waal equals 0.61, 0.93, 1.44, 1.94, 2.92 and \SI{4.85}{\kilo\metre\per\year} at a discharge of 1300, 2000, 3000, 4000, 6000, and \SI{8000}{\metre\cubed\per\second}, respectively, for the new analysis.
The discharge statistics (exceedance curve) has not changed.

\begin{figure}[h]
\includegraphics[width=\columnwidth/2]{figures/GendtseWaard_jaargem.png}
\includegraphics[width=\columnwidth/2]{figures/GendtseWaard_v3_yearavg.png}
\caption{Morphological impact as computed by \dfmi version 3 in backward compatibility mode (left; same as WAQMORF) and in the new default mode (right).}
\label{GendtseWaard_diff}
\end{figure}

The results of the WAQMORF-compatible analysis and the new analysis method are presented side-by-side in \autoref{GendtseWaard_diff}.
The indicated year-averaged equilibrium sedimentation is on average slightly higher than before.
This will have only a minor effect on the estimated sedimentation volume after one year since it is offset by a slightly reduced impacted length.
The impacted length is \SI{1038}{\metre} for the new analysis method\footnote{The length was \SI{1090}{\metre} for the \dfmi versions 3.0.0 until 3.2.0 using a rivers configuration stating celerities of 0.63, 0.97, 1.51, 2.06, 3.16 and \SI{5.35}{\kilo\metre\per\year}.}, compared to \SI{1319}{\metre} using the old analysis method.
Overall, the new method gives a result that closely resembles the results previously obtained using WAQMORF (or the backward compatibility mode of \dfmi).

\section{De Lymen}\label{Sec:DeLymen}

\emph{The input files of this case are included in the distribution under \file{examples/04 - De Lymen}.}

This concerns a section of the "Bedijkte Maas" between Batenburg and Appeltern, along the river Meuse (reach between Grave and Lith).
This case is also used for the backward compatibility tests, see \autoref{Sec:DeLymen_backward}.
The discharge statistics (exceedance curve) has not changed.
The celerity in this reach of the Meuse equals 0.84, 7.80, 8.67, 8.21, 8.28 and \SI{9.56}{\kilo\metre\per\year} at a discharge of 750, 1300, 1700, 2100, 2500, and \SI{3200}{\metre\cubed\per\second}, respectively, for the new analysis.
The values used in the backward compatibility mode depend on the exact characteristic discharges that the user selects for the analysis; more about this later.
The results of the WAQMORF-compatible analysis (based on SIMONA/WAQUA results) and the new analysis method (based on D-Hydro/Delft3D FM results) are presented side-by-side in \autoref{DeLymen_diff}.
There are significant differences in both the magnitude of the year-averaged bed level change and the spatial sedimentation and erosion patterns.
This difference requires a more in-depth analysis.

	\includegraphics[width=\columnwidth/2]{figures/Lymen_jaargem.png}
	\includegraphics[width=\columnwidth/2]{figures/Lymen_v3_yearavg.png}
	\caption{Morphological impact as computed by \dfmi version 3 in backward compatibility mode (left; same as WAQMORF) and in the new default mode (right).}
	\label{DeLymen_diff}
\end{figure}

We start by looking at the sensitivity of the WAQMORF (backward compatibility mode) results.
We consider two cases: in the first case the user specifies a threshold discharge of \SI{1500}{\metre\cubed\per\second} (which was the setting used to obtain the WAQMORF results shown above.) and in the second case the user specifies a threshold discharge of \SI{1250}{\metre\cubed\per\second} and a bank full discharge of \SI{1500}{\metre\cubed\per\second}.
In both cases we assume that the intervention has zero effect until \SI{1500}{\metre\cubed\per\second}, and in both cases a celerity of \SI{9.56}{\kilo\metre\per\year} is used for the high flow condition represented by \SI{1971}{\metre\cubed\per\second}.
The effect of this change is shown in \autoref{DeLymen_waqmorf_diff}.
Note the difference in colour scale.

For the first case, the result ends up being equal to the equilibrium bed level change at \SI{1971}{\metre\cubed\per\second}.
This happens because due to the selected threshold discharge, the \SI{1500}{\metre\cubed\per\second} is treated as a low-flow situation during which a celerity of \SI{0}{\kilo\metre\per\year} is used (effectively switching off the relaxation towards a zero bed level change).
For the second case, the \SI{1500}{\metre\cubed\per\second} condition is treated as the transitional flow regime for which the same celerity of \SI{9.33}{\kilo\metre\per\year} is used as for the high flow regime.
As a result, there is a substantial relaxation to a zero effect during the transitional flow regime and this causes about a factor 2 reduction in the \emph{year-averaged} bed level change.
The spatial pattern of erosion and sedimentation is identical since it is fully determined by the only condition with a non-zero equilibrium bed level change included in the analysis: \SI{1971}{\metre\cubed\per\second}.
The impacted length is also affected by the configuration difference: it is \SI{62}{\metre} for the first case and \SI{144}{\metre} for the second case.
As a result the initial year effect (estimated by integrating the year-averaged bed level change over the impacted length) is about the same (half the effect integrated over double the distance).

Although the user still specifies a threshold discharge for the new \dfmi v3 algorithm, it has no impact on the celerity being used for the various discharge regimes.
The impacted length is now, based on different discharge levels and celerities, computed as \SI{283}{\metre} independent of the threshold discharge selected.\footnote{The length was \SI{195}{\metre} for the \dfmi version 3.2.0 using a rivers configuration stating celerities of 0.60, 5.40, 5.98, 5.50, 5.53 and \SI{6.34}{\kilo\metre\per\year}.
In \dfmi versions older than 3.2.0, the impacted length was computed by integrating the celerity only over discharge conditions above the threshold discharge.
As of version 3.2.0 that integral has been corrected to start from the discharge below which the flow can be considered stagnant, e.g.\ due to barriers.}
Specifying a larger threshold discharge may only reduce the number of simulations to be run.
By definition of the `threshold discharge', the intervention should not have any effect at discharges lower than that value.
Whether zero effect is assumed or derived from actual simulations should not lead to a different final conclusion.

We now continue by looking at the contribution of the various discharge levels to the year-averaged result presented before in \autoref{DeLymen_diff}.
The equilibrium bed level change of the various regimes is shown in \autoref{DeLymen_equilibrium_effects}.
Note that no results are shown for \SI{750}{\metre\cubed\per\second} since at discharge below  \SI{1000}{\metre\cubed\per\second} --- at which the barriers close --- all flow is assumed to be stagnant and hence simulations are not needed.

The sedimentation and erosion patterns change as the discharge varies.
At \SI{2100}{\metre\cubed\per\second} the pattern agrees with the results based on the old WAQMORF approach using solely the SIMONA/WAQUA results of similar discharge \SI{1971}{\metre\cubed\per\second}.
The sedimentation pattern in the main channel remains similar for higher discharges up to \SI{2500}{\metre\cubed\per\second}; the outer bend sedimentation is less pronounced at the highest discharge of \SI{3200}{\metre\cubed\per\second}.
At the lowest simulated conditions (1300 and \SI{1700}{\metre\cubed\per\second}) the intervention (or at least the way it is represented in the model) has a much smaller effect.
The effect is not zero as assumed in the WAQMORF analysis.
The main-channel sedimentation occurs further downstream.
This effect is visible in the year-averaged bed level change obtained using the \dfmi v3 method: the final sedimentation and erosion pattern is a weighted combination of these results.
Although the celerities vary a bit, the weight of each condition is mainly determined by the representative duration of that discharge based on the exceedance statistics.
Since the low-flow conditions occur more frequently, they contribute significantly to the year-averaged results.
This brings us to the effect of discharge peak flattening: for the study presented here, the simulations at \SI{3200}{\metre\cubed\per\second} include this effect (a so-called SD-type simulation) while this effect is not included for the other discharges (so-called S-type simulations).
Since the erosion and sedimentation patterns for \SI{3200}{\metre\cubed\per\second} are very similar to those obtained for \SI{2500}{\metre\cubed\per\second}.
The patterns are even very similar for \SI{2100}{\metre\cubed\per\second} \emph{if we focus on the main channel only} which is the area for which \dfmi is primarily intended.
Therefore, it can be concluded that the effect of peak flattening on the morphological response using the \dfmi method is limited.
Given the uncertainty bounds of the \dfmi analysis, it is not necessary to introduce SD-type simulations for more discharge conditions.

Overall conclusion: the results of the WAQMORF-compatible analysis and the new analysis method show significant differences as can be seen in the side-by-side presentation of the year-averaged bed level changes in \autoref{DeLymen_diff}.
The magnitude of the morphological response obtained from the WAQMORF-analysis depends significantly on the discharge settings chosen for the analysis.
The new analysis method is less sensitive for such choices and it takes into account the impact of the intervention under a wider range of discharge conditions.
Therefore, the new method is preferred and accepted.
